PAHANA EDU Billing System - Help Guide

This help file explains how to use different sections of the billing system.

1. Dashboard

The dashboard provides a quick overview of system statistics:

  • Total Customers: Displays the number of registered customers.
  • Pending Bills: Shows all bills that are not yet paid.
  • Paid Bills: Displays the number of bills already settled.
  • Total Revenue: Calculates the total revenue from all bills.

2. Customer Management

Use this section to manage customer information:

  • Click Add New Customer to register a customer.
  • Use the Search bar to quickly find customers by name or account number.
  • Customer list displays existing customers for editing or deletion.

3. Bill Management

This section is used for generating and managing bills:

  • Click Generate New Bill to create a new bill for a customer.
  • Filter bills by status (All, Pending, Paid).
  • Search bills using customer account numbers.
  • View and update payment status directly from the bill list.

4. Item Management

Items represent products or services included in customer bills:

  • Click Add New Item to register a new product/service.
  • Use the search bar to find specific items.
  • Update or delete items as needed.

5. User Management

Admins can manage system users from this section:

  • Click Add New User to create a new system account.
  • View existing users in the list and manage roles or access rights.

6. Authentication

System security is handled through login sessions:

  • Users must log in with valid credentials to access the system.
  • Authentication tokens are stored locally until logout.
  • Click Logout to securely exit the system.

7. Modal Popups

Forms such as Add Customer, Add Item, or Add User open inside a modal popup window.
